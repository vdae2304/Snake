# Snake

El proyecto consiste en el popular juego del "Snake". 

Se podrá controlar a la serpiente mediante el uso de las 4 flechas 
direccionales. Si se presiona ENTER se pondrá en pausa el juego, al igual que 
de la misma manera se quitará la pausa. 

Conforme se consuma más comida, representadas por una 'O', la serpiente irá 
creciendo de tamaño, al igual que la puntuación aumentará. 

El jugador pierde cuando intenta comerse a sí mismo o choca contra una pared. 
Tras perder la partida, se le pedirá al jugador su nombre y posteriormente 
será redirigido a la tabla de puntuaciones.
