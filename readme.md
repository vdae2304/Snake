# Snake

El proyecto consiste en el popular juego del "Snake". El juego presenta una 
pantalla de título a través de la cual se puede acceder a alguna de las 
siguientes opciones:

- *El juego principal*: El jugador tendrá acceso al juego. Se podrá controlar a 
la serpiente mediante el uso de las 4 flechas direccionales. Si se presiona 
ENTER se pondrá en pausa el juego, al igual que de la misma manera se quitará 
la pausa. Conforme se consuma más comida, representadas por una 'O', la 
serpiente irá creciendo de tamaño, al igual que la puntuación aumentará. El 
jugador pierde cuando intenta comerse a sí mismo o choca contra una pared. Tras 
perder la partida, se le pedirá al jugador su nombre y posteriormente será 
redirigido a la tabla de puntuaciones.
  
- *Tabla de puntuaciones*: Aquí el jugador podrá ver las mejores 10 
puntuaciones que se hayan registrado. Para regresar al menú principal basta con 
presionar cualquier tecla.
